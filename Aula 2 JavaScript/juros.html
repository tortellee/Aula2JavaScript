Faça um programa para calcular o juros simples segundo a fórmula abaixo. Todas as informações devem ser solicitadas ao usuário. J = C*i*n Onde: J = juros, C = capital, i = taxa de empréstimo n = períodos Exemplo: Vamos imaginar o seguinte cenário: um empréstimo de R$ 16.000,00 sobre a taxa de 4% durante 4 meses.
